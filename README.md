# AURA

Android app project.

## Requirements
- Android Studio (latest stable)
- JDK 17+

## Setup
1. Open the project root in Android Studio.
2. Let Gradle sync.

## Build
```bash
./gradlew assembleDebug
```

## Run
Use Android Studio to run on a device or emulator.

## Notes
- `local.properties` is ignored and should point to your Android SDK.
